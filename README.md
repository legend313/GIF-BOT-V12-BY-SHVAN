# GIF Bot 🤖

A simple Discord GIF bot made with Discord.js v12.

## 📌 Features
- Random GIF commands
- Easy prefix system
- Simple command handler

## ⚙️ Commands

- =girl → Random girl GIF
- =boy → Random boy GIF
- =anime → Random anime GIF
- =hug → Hug someone GIF
- =kiss → Kiss GIF
- =help → Show all commands
